The Internet

The Internet is a worldwide network that enables communication using electronic devices.

The Web

The Web is a subset of the internet, in which it has files and documents in the HTML format.

What is the different with the Internet and the Web

The Internet is a network that provides a way to access and share information. The web is more of a service that the internet provides, so we can put files and documents inside.

ASCII and Unicode

ASCII and Unicode are both encoding standards, ASCII is primarily used for the English language, while Unicode has many different characters and is flexible with different languages.

What is UTF-9?

It helps encodes our characters and translates our Unicode characters into binary and vice versa. UTF is short for "Unicode Transformation Format"

URL

"Uniform Resource Locator" is an indentifier that helps us locate a particular resource we are looking for. It contains a domain name and protocol name.

Domain Name

The domain name is an address of a website that enables us to type out alphabets to search for our website.

HTTP

Hypertext Transfer Protocol (HTTP) helps load our website using a Hypertext links. It's an application layer that runs on rules to provide information to website

HTTPS

Similar to HTTP but it adds an extra layer of security and helps protect sensitive information

IP Address

An ip address is a numerical identifier on every device that enables them to connect to the internet.

DNS

DNS or Domanin System name helps translate alphabetical domain names into readable data for machines. It turns it into an IP adddress that makes it easier for idenification.

W3C Recommendations

W3C (World Wide Web Consortium) is described as a community that surrounds the world wide web.This community helps provide Recommendations, that help the web be consistent and usable for other users.

XML, XHTML, HTML, HTML5

XML - Extentible markup language that create rules that help define data typed out

XHTML - A stricter version of XML, has more syntax rules

HTML - A basic scripting language that enables users to typedata

HTML5 - The markup language that is used to structure and present data on the web. It is currently on version 5 thus the number 5.
