Engagement Assignment

1. Block elements take up an entire space of which is available, an example of this is div, p, and many others. Inline tags do the same, but they do not force the an entirely new line to be created. An example of this is span or images.

2. Normal flow the the way pictures and videos are rendered into our code.

3. A float specifies that our object gets taken out of the normal flow. We are able to clear it using the clear id

4. Relative makes sure that it changes the position but stays in the relative flow, absolute removes it from the flow, but is close to its closest element, fixed makes sure that the image or video stays in the position of the page, even if it is moved

5. We can make nice links for mobile devices by making the links have a hover option and thicker area for users to click on.
